#Balloon Popper

##Overview
-------------------------------------------------------------------------------------------------
Simple application that simulates a baloon being inflated until it bursts. Uses the Tutle library
to draw this in the canvas.

##Flow
                                                            yes -> balloon pops
                                                            /
Start -> Draw Balloon -> Up key pressed -> Check if at Max
                                                            \
                                                            no  -> ballon size increases
